Represents an action that can be used when overriding rules in a Managed WAF Ruleset.

Override actions allow you to customize the behavior of individual rules or categories within a managed ruleset without modifying the ruleset itself.

Examples

// Using a known action with IntelliSense
var override = new RuleOverride("rule-id", ManagedWafOverrideAction.Log);

// Using Default to remove a previously set override
var removeOverride = new RuleOverride("rule-id", ManagedWafOverrideAction.Default);

// Using implicit conversion from string for future actions
ManagedWafOverrideAction customAction = "new-action";

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new actions that may be added to the Cloudflare API in the future.

Use the static properties for known actions (e.g., Block, Log) or create custom values using the constructor or implicit string conversion.
