Represents an audit log entry recording an action taken on account resources.

Audit logs are retained for 18 months (v1 API) or 30 days (v2 API). Use filtering parameters to narrow results when querying large datasets.

Remarks

The timestamp location differs between API versions:

  • v1 API: Uses the when field at the top level
  • v2 API: Uses action.time instead
Use the Timestamp property to get the timestamp regardless of API version.
