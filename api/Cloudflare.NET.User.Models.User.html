Represents the authenticated user's profile.

Contains all user information including email, name, contact details, security settings, subscription tier flags, and organization memberships.

Country string

Zipcode string

User's postal/zip code.

Suspended bool

Whether the user account is suspended.

TwoFactorAuthenticationEnabled bool

Whether two-factor authentication is enabled for this user.

TwoFactorAuthenticationLocked bool

Whether two-factor authentication is locked (cannot be disabled). This is typically set by enterprise security policies.

HasProZones bool

Whether the user has any zones on the Pro plan.
