Represents a location hint for R2 bucket placement.

Location hints provide a suggestion for the geographic region where the bucket's data should be stored. Cloudflare will attempt to honor this hint, but data may be placed in a nearby region if the specified location is unavailable.

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new regions that may be added to the Cloudflare API in the future.

Use the static properties for known regions (e.g., EastNorthAmerica) or create custom values using the constructor or implicit string conversion.
