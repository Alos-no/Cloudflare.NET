Represents a DNS record type supported by Cloudflare.

DNS record types define the type of data stored in a DNS record, such as IP addresses (A/AAAA), mail servers (MX), or canonical names (CNAME).

Examples

// Using a known record type with IntelliSense
var type = DnsRecordType.CNAME;

// Creating a DNS record request
var request = new CreateDnsRecordRequest(DnsRecordType.A, "example.com", "192.0.2.1", 3600, true);

// Using implicit conversion from string
DnsRecordType customType = "NEW_RECORD_TYPE";

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new record types that may be added to the Cloudflare API in the future.

Use the static properties for known record types (e.g., A, CNAME) or create custom values using the constructor or implicit string conversion.
