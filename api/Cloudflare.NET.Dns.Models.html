Table of Contents

Namespace Cloudflare.NET.Dns.Models

Classes

BatchDeleteOperation

Delete operation for batch - references a record by its ID.

BatchDnsRecordsRequest

Request to perform batch DNS record operations in a single API call.

Execution Order: Deletes → Patches → Puts → Posts. This order ensures that deletions happen first, allowing re-creation of records with the same name in the same batch.

BatchDnsRecordsResult

Result of a batch DNS records operation.

Contains the results of each operation type. Each array contains the resulting DNS records (for creates/updates) or the deleted records (for deletes).

BatchPatchOperation

Patch operation for batch - includes record ID and fields to update.

Only the fields you include will be updated; other fields retain their values.

BatchPutOperation

Put operation for batch - includes record ID and all fields for full replacement.

This performs a complete replacement of the record. All required fields must be provided.

CreateDnsRecordRequest

Request to create a new DNS record.

Supports all record types and optional features like comments and tags. For MX and SRV records, the Priority field is required.

DnsImportResult

Represents the result of a bulk DNS record import operation.

DnsRecord

Represents a complete DNS record from the Cloudflare API.

This model includes all properties returned by the API, including optional fields like comments, tags, meta information, and record-specific settings.

DnsRecordMeta

Additional metadata for a DNS record.

Contains information about how the record was created and its source.

DnsRecordSettings

Optional settings for DNS records.

These settings control advanced DNS behavior like IPv4/IPv6-only resolution.

DnsScanAcceptItem

A DNS record object for accepting scanned DNS records.

The Cloudflare API requires full DNS record objects (including the Id field) when accepting scanned records. This type provides proper JSON serialization with appropriate ignore conditions for optional fields.

DnsScanReviewRequest

Request to accept or reject scanned DNS records.

Accepted records become permanent DNS records in the zone. Rejected records are discarded from the review queue.

DnsScanReviewResult

Result of a DNS scan review operation indicating how many records were processed.

ListDnsRecordsFilters

Defines the filtering and pagination options for listing DNS records.

PatchDnsRecordRequest

Request to partially update a DNS record (PATCH).

All fields are optional; only include fields you want to change. Unspecified fields retain their current values.

UpdateDnsRecordRequest

Request to fully replace a DNS record (PUT).

All required fields must be provided. This performs a complete replacement of the record at the specified ID.

Structs

DnsRecordType

Represents a DNS record type supported by Cloudflare.

DNS record types define the type of data stored in a DNS record, such as IP addresses (A/AAAA), mail servers (MX), or canonical names (CNAME).
