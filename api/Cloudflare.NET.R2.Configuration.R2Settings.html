EndpointUrl

The R2 S3 API endpoint URL. When set, must contain a format placeholder {0} for the Account ID.

public string? EndpointUrl { get; set; }

Property Value

string

Remarks

When null or empty (the default), the endpoint is automatically computed from Jurisdiction:

  • Default: https://{account_id}.r2.cloudflarestorage.com
  • EU: https://{account_id}.eu.r2.cloudflarestorage.com
  • FedRAMP: https://{account_id}.fedramp.r2.cloudflarestorage.com

When explicitly set, this value takes precedence over Jurisdiction. This allows for custom endpoint configurations or testing scenarios.

Jurisdiction

The jurisdiction for this R2 client configuration. Determines which S3 endpoint to use when EndpointUrl is not explicitly set.

public R2Jurisdiction Jurisdiction { get; set; }

Property Value

R2Jurisdiction

Examples

Configuration via appsettings.json:

{
  "R2": {
    "AccessKeyId": "...",
    "SecretAccessKey": "...",
    "Jurisdiction": "eu"
  }
}

Remarks

Defaults to Default (no jurisdictional restriction), which uses the global endpoint: https://{account_id}.r2.cloudflarestorage.com

When set to a specific jurisdiction (e.g., EuropeanUnion), the S3 endpoint URL is automatically computed to use the jurisdiction-specific endpoint.

If EndpointUrl is explicitly set, it takes precedence over the jurisdiction-based endpoint calculation.

Remarks

Resolution order:

  2. Otherwise, compute from Jurisdiction using GetS3EndpointUrl(string).
