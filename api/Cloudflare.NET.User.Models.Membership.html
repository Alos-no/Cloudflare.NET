Represents a user's membership in a Cloudflare account.

Memberships show which accounts the user has access to and their permissions within each account.

Remarks

User-Scoped: This is the user's view of their account relationships. Each membership grants access to a specific account with defined permissions.

Invitation Workflow: Pending memberships represent invitations that can be accepted or rejected via UpdateMembershipAsync(string, UpdateMembershipRequest, CancellationToken).
