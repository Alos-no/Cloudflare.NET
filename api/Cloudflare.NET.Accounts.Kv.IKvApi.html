Interface IKvApi

Namespace
Cloudflare.NET.Accounts.Kv
Assembly
Cloudflare.NET.dll

Provides access to Cloudflare Workers KV operations. Workers KV is a global, low-latency key-value data store.

public interface IKvApi

Remarks

All operations are account-scoped. The account ID is configured via AccountId.

API Limits:

  • Key name max length: 512 bytes
  • Value max size: 25 MiB
  • Metadata max size: 1024 bytes (serialized JSON)
  • Minimum TTL: 60 seconds
  • Write rate limit: 1 per second per key
