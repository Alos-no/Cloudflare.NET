Turnstile widget mode determining user interaction level.

Widget modes control how the Turnstile challenge is presented to users, ranging from fully invisible to requiring explicit user interaction.

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new modes that may be added to the Cloudflare API in the future.
