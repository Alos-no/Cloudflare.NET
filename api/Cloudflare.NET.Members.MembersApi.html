CreateAccountMemberAsync(string, CreateAccountMemberRequest, CancellationToken)

Invites a new member to the account.

An invitation email will be sent to the specified email address. The user must accept the invitation to become an active member. 

public Task<AccountMember> CreateAccountMemberAsync(string accountId, CreateAccountMemberRequest request, CancellationToken cancellationToken = default)

Parameters

accountId string

The account identifier.

request CreateAccountMemberRequest

The invitation details including email and roles.

cancellationToken CancellationToken

A cancellation token.

Returns

Task<AccountMember>

The created member (with status "pending" until accepted).

Examples

var request = new CreateAccountMemberRequest(
  Email: "developer@example.com",
  Roles: new[] { dnsAdminRoleId }
);
var member = await client.Members.CreateAccountMemberAsync(accountId, request);
Console.WriteLine($"Invited {member.User.Email}, status: {member.Status}");

Remarks

Preview: This operation has limited test coverage.

The email address must not already be associated with an existing member. At least one role must be specified.

Exceptions

ArgumentNullException

Thrown when accountId or request is null.

ArgumentException

Thrown when accountId is empty or whitespace.

DeleteAccountMemberAsync(string, string, CancellationToken)

Removes a member from the account.

The user will immediately lose access to the account's resources. This action cannot be undone - the user must be re-invited to regain access. 

public Task<DeleteAccountMemberResult> DeleteAccountMemberAsync(string accountId, string memberId, CancellationToken cancellationToken = default)

Parameters

accountId string

The account identifier.

memberId string

The membership identifier.

cancellationToken CancellationToken

A cancellation token.

Returns

Task<DeleteAccountMemberResult>

Result containing the deleted membership ID.

Examples

var result = await client.Members.DeleteAccountMemberAsync(accountId, memberId);
Console.WriteLine($"Removed member: {result.Id}");

Remarks

Preview: This operation has limited test coverage.

Warning: You cannot remove yourself from an account. Attempting to do so will result in an error.

Exceptions

ArgumentNullException

Thrown when accountId or memberId is null.

ArgumentException

HttpRequestException

Parameters

The account identifier.

filters ListAccountMembersFilters

Optional filtering and pagination options.

cancellationToken CancellationToken

A cancellation token.

Returns

Task<PagePaginatedResult<AccountMember>>
