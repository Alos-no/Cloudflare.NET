Request to create a new DNS record.

Supports all record types and optional features like comments and tags. For MX and SRV records, the Priority field is required.

Examples

// Create an A record
var aRecord = new CreateDnsRecordRequest(DnsRecordType.A, "www.example.com", "192.0.2.1");

// Create an MX record with priority
var mxRecord = new CreateDnsRecordRequest(DnsRecordType.MX, "example.com", "mail.example.com", Priority: 10);

// Create a record with a comment and tags
var taggedRecord = new CreateDnsRecordRequest(
  DnsRecordType.A, "api.example.com", "192.0.2.2",
  Comment: "API server",
  Tags: ["env:production", "service:api"]
);

Request to create a new DNS record.

Supports all record types and optional features like comments and tags. For MX and SRV records, the Priority field is required. 

public CreateDnsRecordRequest(DnsRecordType Type, string Name, string Content, int Ttl = 1, bool? Proxied = null, string? Comment = null, IReadOnlyList<string>? Tags = null, int? Priority = null, DnsRecordSettings? Settings = null)

Parameters

Type DnsRecordType

The DNS record type.

Name string

The record name (hostname). Can be "@" for the root domain.

Content string

The record content. Format varies by record type.

Ttl int

Time to live in seconds. Default: 1 (automatic).

Proxied bool?

Whether to proxy the record through Cloudflare. Only applies to A, AAAA, and CNAME records.

Comment string

Optional comment describing the record (max 100 characters).

Tags IReadOnlyList<string>

Optional tags in "name:value" format.

Priority int?

Record priority. Required for MX records, optional for SRV records.

Settings DnsRecordSettings

Optional settings for the record.
