Represents the state of a subscription.

Subscription states indicate the lifecycle stage of the subscription:

  • TrialThe subscription is in a trial period
  • ProvisionedThe subscription has been provisioned but not yet paid
  • PaidThe subscription is active and paid
  • AwaitingPaymentThe subscription is awaiting payment
  • CancelledThe subscription has been cancelled
  • FailedA payment or provisioning failure occurred
  • ExpiredThe subscription has expired

var state = SubscriptionState.Paid;

// Using implicit conversion from string

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new states that may be added to the Cloudflare API in the future.
