AccountType

Represents the type of a Cloudflare account.

Account types indicate the tier or category of the account, such as standard or enterprise.

R2Jurisdiction

Represents a jurisdictional restriction for R2 bucket data storage.

Jurisdictions provide a guarantee that objects stored in the bucket will remain within a specific geographic or regulatory boundary. Unlike location hints, jurisdictions are enforced constraints.

R2LocationHint

Represents a location hint for R2 bucket placement.

Location hints provide a suggestion for the geographic region where the bucket's data should be stored. Cloudflare will attempt to honor this hint, but data may be placed in a nearby region if the specified location is unavailable.

R2StorageClass

Represents a storage class for R2 objects.

Storage classes determine the cost and access characteristics of stored objects. Different classes offer trade-offs between storage cost and retrieval latency/cost.
