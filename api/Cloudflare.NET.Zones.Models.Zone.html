Represents the detailed information for a Cloudflare Zone.

A zone represents a domain and its configuration in Cloudflare. This record captures all properties returned by the Cloudflare API for a zone resource.

Parameters

Id string

The unique identifier for the zone (32-character hexadecimal string).

Name string

The domain name of the zone (e.g., "example.com").

Status ZoneStatus

The current status of the zone (e.g., active, pending, initializing).

Account ZoneAccount

The account that owns this zone.

ActivatedOn DateTime?

The timestamp when the zone was activated. Null if not yet activated.

CreatedOn DateTime

The timestamp when the zone was created.

ModifiedOn DateTime

The timestamp when the zone was last modified.

DevelopmentMode int

Seconds remaining in development mode. 0 means development mode is off. When enabled, Cloudflare caching is bypassed.

NameServers IReadOnlyList<string>

The Cloudflare-assigned nameservers for the zone.

OriginalNameServers IReadOnlyList<string>

The nameservers that were on the domain before moving to Cloudflare.

OriginalRegistrar string

The registrar that manages the domain.

OriginalDnsHost string

The DNS host that was hosting the domain before moving to Cloudflare.

Owner ZoneOwner

Information about the user who owns the zone.

Plan ZonePlan

The billing plan for the zone.

Meta ZoneMeta

Additional metadata about the zone.

Paused bool

Whether the zone is paused. When paused, Cloudflare stops proxying traffic.

Permissions IReadOnlyList<string>

The permissions the current API token has on this zone.

Type ZoneType

The type of zone setup (full, partial, or secondary).

VanityNameServers IReadOnlyList<string>

Custom nameservers configured for the zone (Business/Enterprise only).

CnameSuffix string

The CNAME suffix for partial zones.

VerificationKey string

The verification key for the zone.

Tenant ZoneTenant

Tenant information for multi-tenant configurations.

TenantUnit ZoneTenantUnit

Tenant unit information for multi-tenant configurations.

See Also
https://developers.cloudflare.com/api/resources/zones/methods/list/
