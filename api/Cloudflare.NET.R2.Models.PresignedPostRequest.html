Parameters

Key string

The object key (path) where the file will be stored in the bucket.

ExpiresAfter TimeSpan

The duration for which the presigned URL will be valid.

ContentType string

An optional but recommended condition to enforce the MIME type of the uploaded file (e.g., "image/jpeg").

ContentLengthRange (long Min, long Max)?

An optional but recommended condition to enforce the minimum and maximum size of the uploaded file in bytes.

Conditions IEnumerable<S3PostCondition>

An optional collection of custom S3 Post conditions to include in the policy, allowing for advanced validation scenarios.

HeadersToSign IReadOnlyDictionary<string, string>

An optional dictionary of headers to include in the policy, enforcing them on the client-side upload. For POST, these are added as fields in the policy document.
