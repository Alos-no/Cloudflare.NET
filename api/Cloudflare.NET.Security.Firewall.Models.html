AccessRuleMode

Represents the action to take for an IP Access Rule.

Access rule modes define the behavior when a request matches the rule's configuration. Options range from blocking to allowing, with various challenge types in between.

AccessRuleTarget

Represents the target type for an IP Access Rule's configuration.

Access rule targets define what type of network entity the rule applies to, such as a single IP address, IP range, ASN, or country.

LockdownTarget

Represents the target type for a Zone Lockdown rule's configuration.

Zone Lockdown targets define what type of IP entity the rule applies to, such as a single IP address or IP range.

UaRuleMode

Represents the action to take for a User-Agent blocking rule.

User-Agent blocking modes define what happens when a request's User-Agent header matches the rule's configured pattern.
