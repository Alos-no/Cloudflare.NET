Remarks

D1 is Cloudflare's native serverless SQL database built on SQLite. All operations are account-scoped, using the account ID from configuration.

The REST API is subject to global Cloudflare API rate limits and is best suited for administrative operations. For high-throughput query access outside Workers, consider creating a proxy Worker with D1 Worker bindings.

StartImportAsync(string, string, CancellationToken)

Initiates an import operation by requesting an upload URL.

public Task<D1ImportResponse> StartImportAsync(string databaseId, string etag, CancellationToken cancellationToken = default)

Parameters

databaseId string

The database UUID.

etag string

MD5 hash of the SQL file content.

cancellationToken CancellationToken

Cancellation token.

Returns

Task<D1ImportResponse>

Response containing upload_url and filename for SQL file upload.

Remarks

Import workflow:

  1. Compute MD5 hash of your SQL file
  2. Call StartImportAsync(string, string, CancellationToken) with the hash to get upload URL and filename
  3. PUT your SQL file to the upload_url (verify ETag matches)
  4. Call CompleteImportAsync(string, string, string, CancellationToken) with etag and filename to start ingestion
  5. Poll with PollImportAsync(string, string, CancellationToken) until status is "complete"

The database is blocked during import operations.

See Also
https://developers.cloudflare.com/api/resources/d1/subresources/database/methods/import/
