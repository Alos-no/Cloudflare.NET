Represents the action to take for a User-Agent blocking rule.

User-Agent blocking modes define what happens when a request's User-Agent header matches the rule's configured pattern.

var mode = UaRuleMode.Block;

// Using implicit conversion from string

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new modes that may be added to the Cloudflare API in the future.

Use the static properties for known modes (e.g., Block, Challenge) or create custom values using the constructor or implicit string conversion.
