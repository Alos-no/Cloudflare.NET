Represents a jurisdictional restriction for R2 bucket data storage.

Jurisdictions provide a guarantee that objects stored in the bucket will remain within a specific geographic or regulatory boundary. Unlike location hints, jurisdictions are enforced constraints.

Examples

// Create an EU jurisdiction bucket
var bucket = await api.Buckets.CreateAsync("my-bucket", jurisdiction: R2Jurisdiction.EuropeanUnion);

// IMPORTANT: Pass the same jurisdiction for all subsequent operations
var details = await api.Buckets.GetAsync("my-bucket", R2Jurisdiction.EuropeanUnion);
await api.Buckets.SetCorsAsync("my-bucket", corsPolicy, R2Jurisdiction.EuropeanUnion);
await api.Buckets.DeleteAsync("my-bucket", R2Jurisdiction.EuropeanUnion);

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new jurisdictions that may be added to the Cloudflare API in the future.

Important: When to Pass Jurisdiction to API Methods

If you create a bucket with a jurisdiction (e.g., EuropeanUnion or FedRamp), you must pass the same jurisdiction value to all subsequent API operations on that bucket. Without it, the API returns error 10006 "The specified bucket does not exist" even though the bucket exists.

The jurisdiction is required for: GetAsync, DeleteAsync, all CORS operations, all lifecycle operations, all custom domain operations, all managed domain operations, all lock operations, and all Sippy operations.

The jurisdiction is not required for: ListAsync / ListAllAsync (returns all buckets with their jurisdiction in the response) and CreateTempCredentialsAsync (bucket is in the request body).

Best Practice: Store the jurisdiction value when creating a bucket and reuse it for all subsequent operations on that bucket.

When accessing jurisdictional buckets via the S3 API, you must specify the jurisdiction in the endpoint: https://{account_id}.{jurisdiction}.r2.cloudflarestorage.com

The FedRamp jurisdiction requires Enterprise customer status. Contact your Cloudflare account team for access.
