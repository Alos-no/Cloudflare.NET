Represents a Worker route that maps URL patterns to Worker scripts.

Routes determine which requests are handled by which Workers based on URL pattern matching.

Unique route identifier.

Pattern string

Script string

Name of the Worker script to execute for matching requests. Can be null to disable Worker for matching routes.
