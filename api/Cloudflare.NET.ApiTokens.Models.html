Namespace Cloudflare.NET.ApiTokens.Models

Classes

ApiToken

Represents an account-owned API token.

API tokens provide fine-grained access control for Cloudflare API operations. Each token has policies that define what resources and actions are allowed.

CreateApiTokenRequest

Request to create a new API token.

CreateApiTokenResult

Result of creating a new API token, including the secret value.

Important: The Value property contains the token secret and is only available at creation time. Store it securely - it cannot be retrieved again.

CreateTokenPolicyRequest

Policy definition for creating a token.

ListApiTokensFilters

Filtering and pagination options for listing API tokens.

ListPermissionGroupsFilters

Filtering options for listing permission groups.

PermissionGroup

Represents a permission group that can be assigned to API tokens.

Permission groups define what API operations a token can perform. Use GetAccountPermissionGroupsAsync(string, ListPermissionGroupsFilters?, CancellationToken) to discover available groups.

TokenCondition

IP-based access conditions for a token.

Conditions allow restricting token usage based on the source IP address of the API request.

TokenIpCondition

IP address inclusion/exclusion rules for token access.

Supports both allowlist (in) and denylist (not_in) patterns using CIDR notation.

TokenPermissionGroupReference

Reference to a permission group within a token policy.

TokenPolicy

A token policy defining allowed or denied permissions and resources.

Each policy specifies an effect (allow or deny), the permission groups that define what actions are permitted, and the resources the policy applies to.

UpdateApiTokenRequest

Request to update an existing API token.

VerifyTokenResult

Result of verifying a token.

Structs

TokenStatus

Represents the status of an API token.

Token status indicates whether the token is currently usable for API operations.

  • ActiveToken is active and can be used for API requests
  • DisabledToken has been manually disabled
  • ExpiredToken has passed its expiration date
