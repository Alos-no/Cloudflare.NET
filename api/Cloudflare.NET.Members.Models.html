Namespace Cloudflare.NET.Members.Models

Classes

AccountMember

Represents a member of a Cloudflare account.

Account members are users who have been granted access to an account's resources. Each member has one or more roles that define their permissions.

CreateAccountMemberRequest

Request to invite a new member to an account.

An invitation email will be sent to the specified email address. The user must accept the invitation to become an active member.

CreateMemberPolicyRequest

Policy definition for creating or updating a member.

DeleteAccountMemberResult

Result of deleting an account member.

ListAccountMembersFilters

Filtering and pagination options for listing account members.

MemberPermissionGroupReference

Reference to a permission group in a member policy.

MemberPolicy

Access policy for fine-grained member permissions.

Policies provide more granular control than roles by specifying permission groups and the resources they apply to.

MemberResourceGroupReference

Reference to a resource group in a member policy.

MemberUser

User information associated with an account member.

UpdateAccountMemberRequest

Request to update an existing account member.

Only the roles and policies can be updated. The user's email and status cannot be changed through this endpoint.

Structs

MemberStatus

Represents the status of an account member.

Member status indicates the state of the membership:

  • AcceptedThe user has accepted the membership invitation
  • PendingThe invitation is awaiting acceptance
  • RejectedThe user has rejected the invitation
