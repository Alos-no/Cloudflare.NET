Filters for listing Zones.

All filter properties are optional. When multiple filters are specified, they are combined according to the Match parameter. 

public ListZonesFilters(string? Name = null, ZoneStatus? Status = null, string? AccountId = null, string? AccountName = null, int? Page = null, int? PerPage = null, ZoneOrderField? Order = null, ListOrderDirection? Direction = null, FilterMatch? Match = null)

Parameters

Name string

Filter by exact domain name match.

Status ZoneStatus?

Filter by zone status (e.g., active, pending).

AccountId string

Filter by account ID. Use this for multi-account scenarios.

AccountName string

Filter by account name.

Page int?

The page number for pagination (1-based).

PerPage int?

The number of results per page (1-50, default 20).

Order ZoneOrderField?

The field to order results by.

Direction ListOrderDirection?

The sort direction (ascending or descending).

Match FilterMatch?

How to combine multiple filters: "all" (AND) or "any" (OR).
