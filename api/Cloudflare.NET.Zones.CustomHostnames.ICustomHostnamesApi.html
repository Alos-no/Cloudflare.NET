Interface ICustomHostnamesApi

Namespace
Cloudflare.NET.Zones.CustomHostnames
Assembly
Cloudflare.NET.dll

Defines the contract for managing Custom Hostnames (Cloudflare for SaaS) at the zone level.

Custom Hostnames allow SaaS providers to extend Cloudflare's security and performance benefits to their customers' vanity domains.
