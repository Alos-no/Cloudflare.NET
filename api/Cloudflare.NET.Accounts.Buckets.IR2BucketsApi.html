Interface IR2BucketsApi

Namespace
Cloudflare.NET.Accounts.Buckets
Assembly
Cloudflare.NET.dll

Defines the contract for interacting with Cloudflare R2 bucket resources.

This API provides operations for managing R2 buckets including creation, deletion, listing, CORS configuration, lifecycle policies, custom domains, managed domains, bucket locks, and Sippy (incremental migration) configuration. 

public interface IR2BucketsApi

Remarks

Jurisdictional Buckets: Buckets created with a jurisdiction (e.g., EuropeanUnion) require the jurisdiction parameter on all bucket-specific operations. See R2Jurisdiction for details.

Parameters

bucketName string

The name of the bucket.

lifecyclePolicy BucketLifecyclePolicy

The lifecycle policy to apply to the bucket.

jurisdiction R2Jurisdiction?

Required for jurisdictional buckets created with a jurisdiction (e.g., EuropeanUnion).

cancellationToken CancellationToken

A cancellation token.

Returns

Task

A task that represents the asynchronous operation.

Remarks

Lifecycle rules can perform the following actions:

  • Delete objects after a specified age or on a specific date
  • Abort incomplete multipart uploads after a specified age
  • Transition objects to Infrequent Access storage class

Rules are processed within 24 hours of being set. Maximum of 1000 rules per bucket.

https://developers.cloudflare.com/api/resources/r2/subresources/buckets/subresources/lifecycle/methods/update/
