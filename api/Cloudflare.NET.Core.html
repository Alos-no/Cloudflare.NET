Namespace Cloudflare.NET.Core

Classes

ApiResource

Base class for API resource clients, providing shared functionality.

CloudflareApiClientFactory

Factory that creates named ICloudflareApiClient instances. Each named client is configured with its own CloudflareApiOptions and uses a dedicated HttpClient with appropriate authentication and resilience.

CloudflareApiOptions

Represents the configuration options required for the Cloudflare API client.

CloudflareHttpClientConfigurator

Configures HttpClient instances for Cloudflare API communication. Used by both DI registration and dynamic client creation paths to ensure consistent HTTP client configuration.

LoggingConstants

Centralized logging categories for Cloudflare.NET. Use these instead of raw strings to enable consistent, DRY category filtering and avoid typos.

LoggingConstants.Categories
RateLimitingOptions

Configures the automatic retry policy for rate-limited requests (HTTP 429).

ServiceCollectionExtensions

Provides extension methods for setting up the Cloudflare API client in an IServiceCollection.

Interfaces

ICloudflareApiClientFactory

Defines the contract for a factory that creates named ICloudflareApiClient instances. This allows applications to work with multiple Cloudflare accounts or configurations simultaneously.
