Namespace Cloudflare.NET.Subscriptions.Models

Classes

CreateAccountSubscriptionRequest

Request to create a new account subscription.

Creating subscriptions may incur charges. Ensure you have the appropriate billing permissions and understand the pricing of the selected rate plan.

CreateZoneSubscriptionRequest

Request to create a zone subscription (upgrade zone plan).

Use this to upgrade a zone from Free to a paid plan (Pro, Business, Enterprise). Creating subscriptions may incur charges.

DeleteUserSubscriptionResult

Result of deleting a user subscription.

RatePlan

Represents a rate plan for a subscription.

Rate plans define the features and pricing for Cloudflare subscriptions. Each plan has a scope (account, zone, etc.) and may be externally managed.

RatePlanComponent

Component pricing details for a rate plan.

Components define additional features or resources that can be added to a subscription, along with their pricing information.

RatePlanReference

Reference to a rate plan by ID.

Used when creating or updating subscriptions to specify which rate plan to use.

Subscription

Represents a subscription for Cloudflare services.

Subscriptions manage billing plans and add-ons at account, user, or zone level. Each subscription has a rate plan that defines the features included and their pricing.

SubscriptionComponent

Represents a component value in a subscription.

Components are configurable features within a subscription that can have variable quantities or settings, such as additional page rules or workers.

SubscriptionComponentValue

Component value for subscription requests.

Used to set the value of configurable components when creating or updating subscriptions.

UpdateAccountSubscriptionRequest

Request to update an existing account subscription.

Updates may affect billing. Only specified fields will be changed; null fields are left unchanged.

UpdateUserSubscriptionRequest

Request to update an existing user subscription.

Updates may affect billing. Only specified fields will be changed; null fields are left unchanged.

UpdateZoneSubscriptionRequest

Request to update a zone subscription.

Updates may affect billing. Only specified fields will be changed; null fields are left unchanged.

ZoneRatePlan

Represents an available rate plan for a zone.

Rate plans define the pricing and features available for zone subscriptions. Use ListAvailableRatePlansAsync(string, CancellationToken) to discover available plans.

Structs

SubscriptionFrequency

Represents the billing frequency of a subscription.

Subscription frequencies define how often billing occurs:

  • WeeklyBilling occurs every week
  • MonthlyBilling occurs every month
  • QuarterlyBilling occurs every three months
  • YearlyBilling occurs once per year
SubscriptionState

Represents the state of a subscription.

Subscription states indicate the lifecycle stage of the subscription:

  • TrialThe subscription is in a trial period
  • ProvisionedThe subscription has been provisioned but not yet paid
  • PaidThe subscription is active and paid
  • AwaitingPaymentThe subscription is awaiting payment
  • CancelledThe subscription has been cancelled
  • FailedA payment or provisioning failure occurred
  • ExpiredThe subscription has expired
