Represents the status of a Cloudflare Zone.

Zone statuses indicate the current state of a zone in Cloudflare's system, such as whether it's active, pending verification, or has been moved/deleted.

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new statuses that may be added to the Cloudflare API in the future.

Use the static properties for known statuses (e.g., Active, Pending) or create custom values using the constructor or implicit string conversion.
