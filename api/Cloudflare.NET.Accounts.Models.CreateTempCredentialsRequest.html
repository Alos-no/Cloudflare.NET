Parameters

Bucket string

The name of the R2 bucket to scope the credentials to.

The parent access key ID to derive the temporary credentials from.

Permission TempCredentialPermission

The permission level for the temporary credentials.

TtlSeconds int

The time-to-live for the credentials in seconds.

Objects IReadOnlyList<string>

Optional list of specific object keys to scope access to. If not specified, credentials have access to all objects matching the bucket/prefix scope.

Optional list of object key prefixes to scope access to. If not specified, credentials have access to all objects in the bucket.
