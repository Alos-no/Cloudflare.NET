Represents an action that can be performed by a rule in the Cloudflare Ruleset Engine.

Actions define what happens when a rule's expression matches an incoming request. Different actions are available depending on the ruleset phase being configured.

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new actions that may be added to the Cloudflare API in the future.

Use the static properties for known actions (e.g., Block, Challenge) or create custom values using the constructor or implicit string conversion.
