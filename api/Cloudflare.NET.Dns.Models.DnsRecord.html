Class DnsRecord

Namespace
Cloudflare.NET.Dns.Models

Represents a complete DNS record from the Cloudflare API.

This model includes all properties returned by the API, including optional fields like comments, tags, meta information, and record-specific settings.

Examples

var record = await dns.GetDnsRecordAsync(zoneId, recordId);
Console.WriteLine($"{record.Name} ({record.Type}): {record.Content}");

This model includes all properties returned by the API, including optional fields like comments, tags, meta information, and record-specific settings.

Comment = null, IReadOnlyList<string>? Tags = null, int? Priority = null, DnsRecordMeta? Meta = null, DnsRecordSettings? Settings = null)

Parameters

Id string

The unique identifier of the DNS record.

Name string

The record name (hostname).

Type DnsRecordType

The DNS record type.

Content string

The record content (value varies by type: IP address, hostname, text, etc.).

Proxied bool

Whether the record is proxied through Cloudflare.

Proxiable bool

Whether the record can be proxied (depends on type and configuration).

Ttl int

Time to live in seconds. A value of 1 means "automatic" (Cloudflare chooses optimal TTL).

CreatedOn DateTime

When the record was created.

ModifiedOn DateTime

When the record was last modified.

Comment string

Optional comment describing the record.

Tags IReadOnlyList<string>

Optional tags in "name:value" format for organizing records.

Priority int?

Record priority (required for MX records, optional for SRV records).

Meta DnsRecordMeta

Additional metadata about the record, such as whether it was auto-added.

Settings DnsRecordSettings

Optional settings for the record (e.g., IPv4/IPv6-only resolution).

Console.WriteLine($"  Proxied: {record.Proxied} (Proxiable: {record.Proxiable})");
