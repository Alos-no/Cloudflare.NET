Represents the billing frequency of a subscription.

Subscription frequencies define how often billing occurs:

  • WeeklyBilling occurs every week
  • MonthlyBilling occurs every month
  • QuarterlyBilling occurs every three months
  • YearlyBilling occurs once per year

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new frequencies that may be added to the Cloudflare API in the future.
