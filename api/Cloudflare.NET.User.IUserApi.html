Interface IUserApi

Namespace
Cloudflare.NET.User
Assembly
Cloudflare.NET.dll

Defines the contract for interacting with the Cloudflare User API.

This includes retrieving and editing the authenticated user's profile. The User API operates on the currently authenticated user only (self-only access). 

public interface IUserApi

Remarks

All operations in this interface affect only the authenticated user's own profile. There are no endpoints to view or modify other users' profiles.

EditUserAsync(EditUserRequest, CancellationToken)

Edits the authenticated user's profile.

Only the fields provided in the request will be updated. Fields set to null are not modified. 

Task<User> EditUserAsync(EditUserRequest request, CancellationToken cancellationToken = default)

Parameters

request EditUserRequest

The fields to update.

cancellationToken CancellationToken

A cancellation token.

Returns

Task<User>

The updated user profile.

Examples

// Update only first and last name
var updated = await client.User.EditUserAsync(
  new EditUserRequest(
    FirstName: "John",
    LastName: "Doe"));

// Update all editable fields
var updated = await client.User.EditUserAsync(
  new EditUserRequest(
    FirstName: "John",
    LastName: "Doe",
    Country: "US",
    Telephone: "+1-555-555-5555",
    Zipcode: "12345"));

Remarks

Only the following fields can be edited: FirstName, LastName, Country, Telephone, and Zipcode. Email, 2FA settings, and suspension status are managed through separate Cloudflare flows.

RespondToInvitationAsync(string, RespondToInvitationRequest, CancellationToken)

Responds to an invitation (accept or reject).

Warning: This action cannot be undone. Once an invitation is accepted or rejected, its status cannot be changed. 

Task<UserInvitation> RespondToInvitationAsync(string invitationId, RespondToInvitationRequest request, CancellationToken cancellationToken = default)

Parameters

invitationId string

The invitation identifier to respond to.

request RespondToInvitationRequest

The response containing the desired status (accepted or rejected).

cancellationToken CancellationToken

A cancellation token.

Returns

Task<UserInvitation>

The updated invitation with the new status.

Examples

// Accept an invitation
var accept = new RespondToInvitationRequest(MemberStatus.Accepted);
var result = await client.User.RespondToInvitationAsync(invitationId, accept);
Console.WriteLine($"Invitation {result.Status}!");

// Reject an invitation
var reject = new RespondToInvitationRequest(MemberStatus.Rejected);
var result = await client.User.RespondToInvitationAsync(invitationId, reject);

Remarks

Preview: This operation has limited test coverage.

Accepting: Upon acceptance, the user becomes a member of the account with the roles specified in the invitation. The new membership will appear in both the User Memberships API and the Account Members API.

Rejecting: The invitation is marked as rejected. The account admin may send a new invitation.

Exceptions

ArgumentException

Thrown when invitationId is null or whitespace.

ArgumentNullException

Thrown when request is null.

HttpRequestException

Thrown when the API returns an HTTP error status code (404 if not found).

CloudflareApiException

Thrown when the invitation is expired, already responded to, or other API error.

See Also
https://developers.cloudflare.com/api/resources/user/subresources/invites/methods/edit/
