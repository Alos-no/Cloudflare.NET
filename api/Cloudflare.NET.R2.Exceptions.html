Namespace Cloudflare.NET.R2.Exceptions

Classes

CloudflareR2BatchException<T>

Represents an exception thrown during a batch operation (e.g., DeleteObjectsAsync) where some items failed to be processed. This exception contains the list of items that failed, allowing for targeted retries.

CloudflareR2ConfigurationException

Represents an exception thrown when the R2 client is misconfigured (e.g., missing credentials).

CloudflareR2Exception

Represents the base class for exceptions thrown by the Cloudflare.NET.R2 client.

It captures partial metrics accumulated before the failure, allowing for accurate cost tracking even in error scenarios.

CloudflareR2ListException<T>

Represents an exception thrown during a paginated list operation where the operation failed mid-stream. This exception provides access to the data that was successfully retrieved before the failure occurred.

CloudflareR2OperationException

Represents an exception thrown for a failed R2 API operation, such as a 404 on a GetObject call or a 403 on a PutObject call.
