Interface IWorkersApi

Namespace
Cloudflare.NET.Workers
Assembly
Cloudflare.NET.dll

Provides access to Cloudflare Workers API operations.

This interface provides zone-scoped operations for managing Worker routes, which map URL patterns to Worker scripts. Future expansion may include Worker script management, bindings, and other Workers platform features. 

public interface IWorkersApi

Remarks

Worker routes are zone-scoped resources that determine which requests are handled by which Workers based on URL pattern matching.
