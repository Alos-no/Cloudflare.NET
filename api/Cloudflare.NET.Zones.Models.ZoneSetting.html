Represents a zone configuration setting.

The Value property contains the setting's current value, which varies by setting type. Use JsonElement methods to extract the appropriate type (GetString, GetInt32, etc.).

Parameters

Id ZoneSettingId

The setting identifier (e.g., MinTlsVersion, Ssl).

Value JsonElement

The current value of the setting. Type varies by setting:

  • Toggle settings: string "on" or "off"
  • Numeric settings: integer (e.g., TTL in seconds)
  • Selection settings: enumerated string values
  • Complex settings: nested JSON object
Editable bool

Whether this setting can be modified for the current zone/plan.

ModifiedOn DateTime?

When the setting was last modified.
