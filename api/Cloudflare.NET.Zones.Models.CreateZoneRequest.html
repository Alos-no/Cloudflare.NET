Request to create a new Zone in Cloudflare.

Creates a zone for a domain and associates it with an account. The domain must not already be added to Cloudflare in the same or another account.

Request to create a new Zone in Cloudflare.

Creates a zone for a domain and associates it with an account. The domain must not already be added to Cloudflare in the same or another account. 

public CreateZoneRequest(string Name, ZoneType Type, ZoneAccountReference Account, bool? JumpStart = null)

Parameters

Name string

The domain name to add (e.g., "example.com").

Type ZoneType

The type of zone setup: Full for Cloudflare-hosted DNS, Partial for CNAME setup, Secondary for secondary DNS.

Account ZoneAccountReference

The account to associate the zone with.

JumpStart bool?

Whether to automatically fetch existing DNS records from the domain. Defaults to false. This is a best-effort operation; failure does not prevent zone creation.
