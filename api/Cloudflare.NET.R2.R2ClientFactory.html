Factory that creates named IR2Client instances. Each named client is configured with its own R2Settings and uses a dedicated AmazonS3Client with appropriate credentials.

Remarks

This factory caches created clients to avoid the overhead of creating new S3 clients on every request. Clients are cached by name and reused for subsequent requests.

Named clients are registered using AddCloudflareR2Client(IServiceCollection, string, Action<R2Settings>) .
