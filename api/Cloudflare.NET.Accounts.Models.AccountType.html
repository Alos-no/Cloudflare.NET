Represents the type of a Cloudflare account.

Account types indicate the tier or category of the account, such as standard or enterprise.

Remarks

This is an extensible enum that provides IntelliSense for known values while allowing custom values for new account types that may be added to the Cloudflare API in the future.

Use the static properties for known types (e.g., Standard, Enterprise) or create custom values using the constructor or implicit string conversion.
